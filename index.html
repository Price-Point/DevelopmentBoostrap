PricePoint Developer Bootstrap

PricePoint dev machine setup

Install latest version of VirtualBox

Download latest stable version of Lubuntu and create new VM with at least 200gb disc, 4 cores, 16gb ram, and 64mb video memory. Be sure to also update clibboard and drag and drop settings to bicirectional.

Install guest extensions as described here

Run the following in a terminal window and follow the remaining prompts

wget -qO - https://api.github.com/repos/Price-Point/development-bootstrap/contents/bootstrap.sh --header 'Accept: application/vnd.github.v3.raw' | bash
